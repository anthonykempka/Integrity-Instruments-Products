Data logger

To select the data logger, first select the tab for the type of module that you have. Then under the tools menu select data logger. The data logger window will open.

Set the interval time you wish to take samples. You can go to approximately .06 seconds dependent on the module. If a time is selected that is too short a window will open telling you that.

Now you must select a file the logging samples will be stored in. Click the browse button and select a file, or name a file and location it will be placed. Click save and the window will close.

Now you must select the Unipolar or Bipolar channels to be logged. Clicking in the selection window will place a check mark on the channels to be sampled. Note that if you select the Bipolar channel, the corresponding Unipolar channels will be deleted. You click the Bipolar off box to restore the Unipolar channels.

In the USB module you can also select the gain for each analog channel.

To start the logging click the “start” button. To stop the logging click the “halt” button.

When started the logging values will be displayed on the graph and stored in the file you selected.

To “zoom in” on the graph change the numbers in the “Minimum value” and “Maximum value” boxes. The graph will change with the values you have selected.

With the USB module you can also data log with the USB stream mode. To implement this function under the tools menu click the “stream mode”, which will open the USB stream mode window. The graph will be scaled for whatever reference is set in the “ADC Reference” selection box for the USBM400 module screen. You can also record the readings if you enable the logging file box, and designate a file name. When you click the “start” button the stream mode will begin. Click the “Stop” button will halt the operation.
