Getting Started

 

The Data Acquisition Module utility is applicable for all of the data acquisition modules offered by Integrity Instruments. All modules with this utility have data logging capability.

 

Select the tab for the type of unit you have.

 

There are control boxes on the screen you select. The control boxes and their function are:

 

Read Current       Reads the current settings on the unit.

Configure             Sets the unit to the parameters you have selected.

Defaults                 Sets the factory defaults to the parameter settings. If you do a Configure now the unit will be set to factory defaults.

Comm Setup         Select the comm port of your PC and the baud rate.

There is no Comm Setup for USB modules

 

NOTE:

 

After configuring your module, to make your parameter settings active, you must re-cycle power to the module.
