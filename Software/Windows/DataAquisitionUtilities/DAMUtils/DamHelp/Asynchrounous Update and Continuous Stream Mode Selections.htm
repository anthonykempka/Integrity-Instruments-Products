Asynchronous update and Continuous stream mode selections.

You can select to send the counter and the inputs status to be sent back in these modes.

You select the number of analog samples to be sent back. You must then select the sample type, Bipolar or Unipolar.

The sample modes box contains the type of sample. The configuration box contains the selected samples.

Highlight a selection in the configuration box and click the “delete” box to delete it.

Highlight a selection in the Sample mode box and click the “add” box to add it to the configuration box.

Highlight a selection in the configuration box and click the “up” or “down” box to move its position.

The configuration box will only allow the number of samples to be taken as the unit will allow.

For the USB module you can also select a gain for each channel. Highlight the channel in the configuration box. Now by highlight the number in the gain box, that gain is for that channel. Select another channel in the configuration box and repeat the procedure.

You can select the same analog sample in the configuration box. For example in the 232M300 module you could select CH0 eight times.
