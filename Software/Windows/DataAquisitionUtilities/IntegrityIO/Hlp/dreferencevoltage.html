double dReferenceVoltage

A/D & D/A Reference voltage

Value 5.0 by default

 

Syntax

object.dReferenceVoltage = value

or

value = object.dReferenceVoltage

Remarks:

Property has write only attributes.

This value is set to 5.0 volts by default. If your module has a diffarent voltage refrence in it, this value must be set accordingly. Failure to do so will yeild incorrect A/D and D/A convertions.
