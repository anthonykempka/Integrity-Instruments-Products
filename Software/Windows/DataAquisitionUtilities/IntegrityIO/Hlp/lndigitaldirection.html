long lnDigitalDirection

Digital I/O port pin direction

 

Syntax

object.lnDigitalDirection = value

or

value = object.lnDigitalDirection

Remarks:

Property has read/write attributes.

Value = 0xAABB

AA = Port 1, BB = Port2

(See ADC module documetation for hardware specific details).
