SetDIOBit

Output a single bit to the digital IO Port.

Syntax

value = object.GetDIOBit(long bitValue, long bitPosition) as Long

bitValue: A data 0 or 1 will be written out to the data port.

bitPositon: Valid bit positions are 0 ... 15. Bit positions are weighted as XXYY where XX = Port 2, and YY = Port 1. Bit position 0 refers to Port1 bit 0, Bit position 8 refers to Port 2 Bit 0.

Remarks

This function implements a read/modify/write on the digital I/O port.
