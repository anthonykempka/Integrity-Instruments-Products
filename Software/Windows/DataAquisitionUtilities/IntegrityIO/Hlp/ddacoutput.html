double dDACOutput

D/A output voltage

Value = 0 ... +vRef

vRef = dRefrenceVoltage

Syntax

object.dADCUnipolar = value

Dependencies

lnDACControl, dRefrenceVoltage

Remarks:

Property has write only attributes.

The lnDACControl word must be set prior to reading this property. Failure to set this property may result in unpredicable results.
