long lnComPort

Comport configuration

 

Syntax

object.lnComPort = value

or

value = object.lnComPort

Remarks:

Property has read/write attributes.

Any changes to this property modify the communication parameters immediately. It is not neccessary to close and reopen the port after writing this value.

Identifies Commport

Values: 1 ... 255
