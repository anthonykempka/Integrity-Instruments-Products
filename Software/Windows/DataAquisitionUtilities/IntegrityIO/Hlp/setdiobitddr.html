SetDIOBitDDR

Output a single bit to the digital IO port data direction register.

Syntax

value = object.GetDIOBitDDR(long bitDirection, long bitPosition) as Long

bitDirection: Writes a 0 or 1 to the appropriate bit position in the data direction register. Writing a data 0 will force the corresponding port pin to be an output, writing a data 1 makes the port pin an input.

bitPositon: Valid bit positions are 0 ... 15. Bit positions are weighted as XXYY where XX = Port 2, and YY = Port 1. Bit position 0 refers to Port1 bit 0, Bit position 8 refers to Port 2 Bit 0.

Remarks

This function implements a read/modify/write on the digital I/O port.
