BOOL bPWMUpdate

Update the PWM registers.

Syntax

object.bDialogDisable = value

Dependencies

lnPWMFreq, lnPWMDuty

Remarks:

Property hasWrite only attributes.

Writing any value to this property will cause the control to construct the "P" command (See hardware users manual) and send it to the unit. This function uses the current settings of the lnPWMFreq and lnPWMDuty properties.
