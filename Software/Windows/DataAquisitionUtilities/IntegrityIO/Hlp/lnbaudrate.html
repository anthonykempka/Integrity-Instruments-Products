long lnBaudRate

Commport communication rate configuration.

 

Syntax

object.lnBaudRate = value {9600, 19200, 57600, or 115200}

or

value = object.lnBaudRate

Remarks:

Property has read/write attributes.

Writing to this value will update the communication rate immediatly. It is not neccesary to close the reopen communications.

Only the following baud rates are supported.

9600, 19200, 57600, 115200
