long lnADCControl

A/D control nibble

 

Syntax

object.lnADCControl = value

or

value = object.lnADCControl

Remarks:

Property has read/write attributes.

Hexadecimal values are as shown below.

0 - Diffarential: CH0+ CH1-
1 - Diffarential: CH2+ CH3-
2 - Diffarential: CH4+ CH5-
3 - Diffarential: CH6+ CH7-
4 - Diffarential: CH0- CH1+
5 - Diffarential: CH2- CH3+
6 - Diffarential: CH4- CH5+
7 - Diffarential: CH6- CH7+

8 - Single Point: CH0
9 - Single Point: CH2
A - Single Point: CH4
B - Single Point: CH6
C - Single Point: CH1
D - Single Point: CH3
E - Single Point: CH5
F - Single Point: CH7
