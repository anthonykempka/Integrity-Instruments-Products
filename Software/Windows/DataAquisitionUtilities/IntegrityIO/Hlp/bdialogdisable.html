BOOL bDialogDisable

Set or Clear the dialog disable flag.

Syntax

value = object.bDialogDisable

or

object.bDialogDisable = value

Remarks:

Setting this flag to a value of TRUE (1) will disable any pop up dialog boxes when errors occur.

Default value is FALSE (0)

Property has Read/Write attributes.
