In order to get up an running quickly with the Integriy Instruments Data Aquisition and Control modules, we have compiled a set of examples for some commonly used development platforms. All of these examples perform the same set of tasks. All of the examples assume that the ADC-xxx has a refrence voltage of 5.0 volts, and that the unit is connected to COM1.

1.Output a voltage between 0 and +Vref using Analog Out A.
2. Read back the voltage from Analog Out A on Analog In Channel 0.
3. Configure one digital port for output.
4. Configure one digital port for input.
5. Toggle the state of the digital output pin, read it back and display it.
