GetDIOBitDDR

Read a single bit from the digital IO port data direction register.

Syntax

value = object.GetDIOBitDDR(long bitPosition) as Long

GetDIOBit returns a 0 or 1 long value that represents the state of the Digitial I/O data direction register. Valid bit positions are 0 ... 15. Bit positions are weighted as XXYY where XX = Port 2, and YY = Port 1. Bit position 0 refers to Port1 bit 0, Bit position 8 refers to Port 2 Bit 0.

Remarks

Refer to the users manual for specific details. This function implements the "T" command.
