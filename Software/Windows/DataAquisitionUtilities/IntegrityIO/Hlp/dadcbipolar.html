double dADCBipolar

Sample A/D bipolar voltage

Value = -vRef ... +vRef

vRef = dRefrenceVoltage

Syntax

value = object.dADCBipolar

Dependencies

lnADCControl, dRefrenceVoltage

Remarks:

Property has read only attributes.

The lnADCControl word must be set prior to reading this property. If it is not, the result may be unreliable.
