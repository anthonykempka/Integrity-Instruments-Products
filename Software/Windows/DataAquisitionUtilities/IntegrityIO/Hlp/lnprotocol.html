long lnProtocol

Target Module Communication Protocol

 

Syntax

value = object.lnProtocol

or

object.lnProtocol = value

Remarks:

Property has read/write attributes.

Properties identifies the target module communication protocol to use

1 = RS-232
2 = RS-485
3 = USB

Specific protocolse are not implemented in version 1.0 of the software software.
