BOOL bPortOpen

Open/Close the serial port.

Syntax

value = object.bPortOpen

or

object.bPortOpen = value

Dependencies

lnComPort, lnBaudRate

Remarks:

Setting this flag to a value of TRUE (1) will force the object to attempt an open of the serial port defined by the lnComPort property. It will use the baud rate defined by the lnBaudRate property.

If the port open fails reading this propery will return a FALSE(value)

Default value is FALSE (0)

Property has Read/Write attributes.
