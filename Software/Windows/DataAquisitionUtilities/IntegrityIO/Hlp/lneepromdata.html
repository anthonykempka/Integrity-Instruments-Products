long lnEEPROMData

EEPROM data

 

Syntax

object.lnEEPROMData = value

or

value = object.lnEEPROMData

Dependencies:

lnEEPROMAddress

Remarks:

Property has read/write attributes.

Causes data to be read from or written to EEPROM. The lnEEPROMAddress must be set prior to reading or writing this property.
