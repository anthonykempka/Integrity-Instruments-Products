Integrity ActiveX I/O Module Documentation

Properties Methods Examples

 

The Integrity IO module provides the communications and protocol interface for your application. This currently contains code that will allow your application access to all of the input and output functions of the Integrity Instruments ADC-xxx modules through a simple, easy to use interface.

If you are reading this document, then and ActiveX control (IntegrityIO.ocx) has been installed and registered on your computer. This ActiveX control, like most ActiveX controls, may be used for adding ADC-xxx functionality to your Visual Basic, Visual C++, Excel, and Test Point programs.
